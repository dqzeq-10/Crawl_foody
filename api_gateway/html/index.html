API GATEWAY: Foody Crawler

Các Endpoint Có Sẵn

GET /api/db/branches

Lấy danh sách tất cả các chi nhánh.

GET /api/db/branch/{branch_id}

Lấy thông tin chi tiết của một chi nhánh cụ thể theo ID.

POST /api/db/branch

Tạo một chi nhánh mới.

PUT /api/db/branch/{branch_id}

Cập nhật thông tin chi nhánh hiện có.

DELETE /api/db/branch/{branch_id}

Xóa một chi nhánh theo ID.

GET /health

Endpoint kiểm tra trạng thái để xác minh xem Cổng API có đang hoạt động không.

Cách Sử Dụng

Thực hiện các yêu cầu HTTP đến các endpoint ở trên bằng cách sử dụng các phương thức HTTP và tham số thích hợp. Tất cả các phản hồi đều ở định dạng JSON. Nhấp vào bất kỳ liên kết endpoint nào để truy cập trực tiếp.
