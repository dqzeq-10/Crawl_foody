API GATEWAY: Foody Crawler

Các Endpoint Có Sẵn

API CRAWLER Docs

GET /api/crawler/status

Lấy trạng thái hiện tại của dịch vụ crawler.

POST /api/crawler/crawl

Bắt đầu một công việc crawl mới. Tham số tùy chọn: pages.

POST /api/crawler/reset

Đặt lại trạng thái crawler (dùng cho debug/kiểm thử).

API DATA INGESTION Docs

GET /api/ingestion/status

Lấy trạng thái hiện tại của quá trình xử lý dữ liệu.

POST /api/ingestion/process

Bắt đầu quá trình xử lý dữ liệu mới. Tham số tùy chọn: force.

GET /api/ingestion/preview

Xem trước dữ liệu đã xử lý. Tham số tùy chọn: limit.

POST /api/ingestion/reset

Đặt lại trạng thái xử lý dữ liệu (dùng cho debug/kiểm thử).

GET /api/ingestion/db/check

Kiểm tra kết nối đến database API.

API DATA INGESTION Docs

GET /api/crawler/schedule

Lấy danh sách tất cả lịch trình crawl đã cấu hình.

POST /api/crawler/schedule

Tạo một lịch trình crawl mới. Yêu cầu body JSON với các trường: name, cron_expression, pages (tùy chọn), description (tùy chọn), active (tùy chọn).

Ví dụ cron: "0 0 * * *" (chạy lúc 0:00 mỗi ngày)

PUT /api/crawler/schedule/{name}

Cập nhật lịch trình crawl hiện có.

DELETE /api/crawler/schedule/{name}

Xóa lịch trình crawl.

POST /api/crawler/schedule/{name}/toggle

Bật/tắt lịch trình crawl.

API BRANCHES Docs

GET /api/db/branches

Lấy danh sách tất cả các chi nhánh.

GET /api/db/branch/{branch_id}

Lấy thông tin chi tiết của một chi nhánh cụ thể theo ID.

POST /api/db/branch

Tạo một chi nhánh mới.

PUT /api/db/branch/{branch_id}

Cập nhật thông tin chi nhánh hiện có.

DELETE /api/db/branch/{branch_id}

Xóa một chi nhánh theo ID.

GET /api/db/branch/search

Tìm chi nhánh theo các ký tự nhập vào.

GET /health

Endpoint kiểm tra trạng thái để xác minh xem Cổng API có đang hoạt động không.

Foody Web Page

GET /foody

Trang web giao diện người dùng Foody, hiển thị và tìm kiếm thông tin các chi nhánh.

Cách Sử Dụng

Thực hiện các yêu cầu HTTP đến các endpoint ở trên bằng cách sử dụng các phương thức HTTP và tham số thích hợp. Tất cả các phản hồi đều ở định dạng JSON. Nhấp vào bất kỳ liên kết endpoint nào để truy cập trực tiếp.

Dịch Vụ

  • DB API: Cung cấp các thao tác cơ sở dữ liệu cho các chi nhánh Foody
  • Web Page: Giao diện người dùng để duyệt và tìm kiếm các chi nhánh Foody
